# GameLog

GameLog is a desktop application designed to help users manage and track their video game library, backlog, and progress. The project integrates external game data to provide rich information and a streamlined user experience.
This project was developed as part of the course 'Database Technology' during the 7th semester at dpt. of Information and Electronics Engineering at International Hellenic University.

---

## Features

- Gaming backlog CRUD operations
- Favorites list
- Progress tracking upon the backlog list
- notifications
- dark mode support
- User account CRUD operations

---

## Technologies Used

- Java  
- JavaFX  
- Maven  
- PostgreSQL  
- REST API  
- FXML
